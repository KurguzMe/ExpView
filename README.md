# 👁️ ExpView
**ExpView** — this is a lightweight, convenient Minecraft mod that adds an experience point counter to the screen and a menu where you can move the counter with the mouse and also convert experience.

---

## Download
The latest and most stable versions of the mod are always available on the official page:
* **[Download the mod on Modrinth](https://modrinth.com/mod/expview#download)**

---

## 🐛 Found a bug or have an idea?? / Issues
If you encounter issues, the game crashes, or you want to suggest a new feature:
1. Go to the tab **[Issues](https://github.com/KurguzMe/ExpView/issues)** at the top of this page.
2. Click the button **New Issue**.
3. Select the appropriate template (Bug Report) and fill out the form following the instructions.

*You can also attach log files (`latest.log`) when reporting game crashes!*
